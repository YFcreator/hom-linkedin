GUIDA SETUP N8N
1. Workflow "Pubblica Post" (Pagina HOM)
Trigger: Webhook (POST) → LinkedIn: Create Post (con OAuth2, Organization URN: 107656085) → Respond to Webhook

2. Workflow "Pubblica Post" (Profilo Personale)
Trigger: Webhook (POST) → LinkedIn: Create Post (con OAuth2, come persona) → Respond to Webhook

3. Workflow "Leggi Metriche"
Trigger: Webhook (POST) → LinkedIn: Get Post Stats → Respond to Webhook

4. Workflow "Coda Post" (Gestione Google Sheet)
Trigger: Webhook (POST) → IF action="queue": Google Sheets: Append Row (id, text, topic, scheduledDate, brand, label, status="pending") / IF action="cancel": Google Sheets: Delete Row (cerca per id) → Respond to Webhook {success: true}

5. Workflow "Pubblica Automatico" (Cron)
Trigger: Schedule (ogni 15 min) → Google Sheets: Read All Rows → Filter: scheduledDate ≤ NOW() AND status="pending" → Per ogni riga: IF brand="personal" → chiama webhook Pubblica Personale, ELSE → chiama webhook Pubblica HOM → Google Sheets: Update Row (status="published") → (opzionale) Delete Row dal foglio

Google Sheet — colonne richieste:
id | text | topic | scheduledDate | brand | label | status

Sicurezza: Header Auth su OGNI workflow
In ogni workflow N8N, dopo il nodo Webhook, aggiungi un nodo IF che verifica:
{{ $json.headers['x-webhook-secret'] }} === "IL_TUO_SECRET"
Usa lo stesso secret configurato sopra. Se non corrisponde → Respond to Webhook con errore 403. Questo impedisce a chiunque conosca l'URL di usare i tuoi webhook.
