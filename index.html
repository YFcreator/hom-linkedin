LinkedIn via N8N
Collega N8N per pubblicare direttamente su LinkedIn e leggere le metriche in automatico. Crea 2 workflow in N8N con trigger Webhook, poi incolla qui le URL.
Riceve JSON: { text, scheduledDate, topic }. Deve restituire: { success, linkedinUrl, postUrn }
Riceve JSON: { postUrn }. Deve restituire: { impressions, reactions, comments, shares }
Non configurato
GUIDA SETUP N8N
1. Workflow "Pubblica Post"
Trigger: Webhook (POST) → LinkedIn: Create Post (con OAuth2) → Respond to Webhook
Il nodo LinkedIn riceve il campo text dal body e pubblica come post della pagina aziendale HOM.
Risposta: { "success": true, "linkedinUrl": "...", "postUrn": "urn:li:share:..." }

2. Workflow "Leggi Metriche"
Trigger: Webhook (POST) → LinkedIn: Get Post Stats → Respond to Webhook
Riceve postUrn, interroga l'API LinkedIn per le stats del post.
Risposta: { "impressions": 1200, "reactions": 45, "comments": 12, "shares": 8 }

3. Notifiche (opzionale)
Aggiungi un terzo workflow con Cron trigger che controlla i post pianificati e ti manda una notifica Telegram/email quando è ora di pubblicare.
